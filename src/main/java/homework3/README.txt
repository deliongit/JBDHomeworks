Управление транспортными средствами (TransportManagement.java)

Описание:
Программа представляет собой консольное приложение, моделирующее изготовление, управление и утилизацию различных типов транспортных средств.
Поддерживается пять типов транспорта: 
- Автомобиль;
- Грузовик;
- Вертолёт;
- Корабль;
- Подводная лодка.

Каждое транспортное средство обладает уникальными характеристиками, состояниями и действиями. 
Основные возможности программы:
- Создание транспорта вручную - выбор типа, марки и максимальной скорости.
- Создание случайного транспорта - автоматическая генерация с реалистичными марками и скоростями.
- Управление транспортом - выполнение доступных действий (запуск двигателя, движение, повороты и другие).
- Просмотр списка всех транспортных средств.
- Детальная информация - отображение текущего состояния транспортного средства (скорость, двигатель, фары, якорь, глубина погружения и прочие).
- Утилизация транспорта - удаление из списка.
 
Архитектура и компоненты 
Vehicle - абстрактный базовый класс для всех транспортных средств.
Engine - иерархия двигателей (CombustionEngine, ElectricEngine, JetEngine, NuclearEngine).
ControlSpeed - управление текущей и максимальной скоростью.
States - перечисления состояний (двигатель, фары, окна, якорь и другие).
Action - перечисление всех возможных действий.
VehicleValidator и StatusValidator - система проверки допустимости действий в зависимости от текущего состояния транспортного средства или его компонента.
UserInterface - реализация консольного меню и взаимодействия с пользователем.
 
Особенности кода:
- Консольная навигация с обработкой ошибок ввода.
- Все действия проходят валидацию состояний, чтобы избежать некорректных операций (например, нельзя включить фары дважды или бросить якорь, если он уже брошен).
- Использованы запечатанные классы для наследования контроля.
- Активно применено использование перечислений.
- Применены паттерны, такие как: наследование, инкапсуляция, полиморфизм.
